CsvHelper.ICsvWriter.ClearRecordCache
void ClearRecordCache()
Clears the record cache for all types. After WriteRecord{T} is called the first time, code is dynamically generated based on the CsvPropertyMapCollection, compiled, and stored for the given type T. If the CsvPropertyMapCollection changes, ClearRecordCache() needs to be called to update the record cache.
