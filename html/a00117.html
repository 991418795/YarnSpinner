void CsvHelper.ICsvReaderRow.ClearRecordCache ( Type  type)

Clears the record cache for the given type. After ICsvReaderRow.GetRecord{T} is called the first time, code is dynamically generated based on the CsvPropertyMapCollection, compiled, and stored for the given type T. If the CsvPropertyMapCollection changes, ClearRecordCache( Type ) needs to be called to update the record cache.

Parameters
typeThe type to invalidate.
